# AI HQ Command Center — Interactive 3D Office Dashboard

> An interactive 3D isometric office dashboard featuring AI agent simulation, sprint tracking, standup mode, and real-time team stats.

[View on Claude AI](https://claude.ai/public/artifacts/7f72a830-90fd-4c3a-860b-eda9c80cce16)

## Description

This project is an AI-powered HQ Command Center built as a single-file HTML/CSS/JavaScript application. It renders an interactive isometric 3D office floor where AI agents move, collaborate, and complete tasks in real time.

## Features

- **Interactive 3D Isometric Office** — Navigate a fully rendered office environment with agents, desks, break rooms, and a lab
- **AI Agent Simulation** — Multiple agents (Luke, Allan, Ben, Cory, Andrew, Susie) move autonomously and report status
- **Standup Mode** — Trigger daily standups and watch agents present their updates
- **Sprint Board** — Track active sprint tasks with assignees and progress
- **Real-Time Team Stats** — Tasks done today, in-progress count, and blockers
- **Agent Log** — Live feed of agent activity and system events
- **Multiple Floors** — Switch between Main Floor, Lab, and Break Room
- **Time of Day** — Dynamic lighting based on time
- **Party Mode & Force Standup** — Fun interactive triggers

## Tech Stack

- HTML5 / CSS3
- Vanilla JavaScript (Canvas 2D rendering)
- No external dependencies
